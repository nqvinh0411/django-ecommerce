Hướng dẫn sử dụng API
Bắt đầu với API

Để sử dụng API của hệ thống E-commerce, bạn cần thực hiện các bước sau:

1. Xác thực

Hầu hết các API yêu cầu xác thực bằng JWT token. Để lấy token, gọi API đăng nhập:

POST /api/v1/auth/login/
{
    "username": "your_username",
    "password": "your_password"
}
2. Sử dụng token

Thêm token vào header của request:

Authorization: Bearer your_jwt_token
3. Định dạng response

Tất cả API trả về kết quả theo định dạng chuẩn:

{
    "status": "success" | "error",
    "status_code": 200 | 4xx | 5xx,
    "message": "Thông báo",
    "data": {...} | [...],
    "pagination": {
        "count": 100,
        "page_size": 10,
        "current_page": 1,
        "total_pages": 10,
        "next": "URL trang tiếp theo",
        "previous": "URL trang trước"
    }
}
